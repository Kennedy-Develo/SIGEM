# SIGEM

Sistema de Gestão de Manifestações.

## Tecnologias

- Backend: PHP com Laravel
- Frontend: Vue 3
- Banco de dados: SQLite inicialmente

## Ordem de desenvolvimento

1. Estrutura e identidade visual
2. Autenticação
3. Usuário administrador
4. Gestão de usuários e permissões
5. Módulo de manifestações
